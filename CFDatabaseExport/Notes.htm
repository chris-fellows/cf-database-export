CFDatabaseExport

Utility for exporting data from any OLEDB database to CSV, HTML, XML, Excel or a SQL script. The user selects the SQL query and output format and the data is exported in the required format.

Output formats

Format Comments
CSV file Output each resultset returned by the SQL query to a separate CSV file.
HTML Takes a template HTML file and inserts an HTML TABLE element with each resultset returned by the SQL query.
XML Exports all rows to an XML file.
Excel
JSON Exports all rows to JSON format.
SQL script Takes two SQL templates, one for the main part of the script and one for each row and produces a new SQL script. The main template requires a placeholder where the SQL for each row will be inserted.

For example:
  • Main script creates a temporary table, defines the placeholder for the rows, and then the SQL for processing the temporary table and writing the data to a database table.
  • The row template takes selected columns from the resultset and inserts the row in to the temporary table defines in the main script.
Any format Takes an XSL transform file and transforms the resultset. For example, Transform to an XML or HTML file.


Note: I developed a similar tool while working for a previous employer. (It was actually developed from my own idea on my own time because I was spending so much time responding to data requests and having to write a new query each time.) CFDatabaseExport was written from scratch on my own time and it does not use any of the original code.
